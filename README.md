my-sport-life
=============

Running, rowing, biking, skiing, swimming... day by day.

Design
------
* Data are written in/converted from existing sources in YAML.
* NICE: web interface/desktop tool (console) for data management
  ensures validity.
* Simple validation tool that performs also YAML to JSon conversion.
* My GDocs log like HTML interface that loads JSon file(s) from
  a given URL/path. One HTML interface (not generated), data source
  anywhere.
